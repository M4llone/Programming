# Лабораторная работа: Paint на C++ с использованием SFML

Основная задача - дополнить недостающие функции, освоив базовые конструкции языка  C++, алгоритмы и контейнеры STL, а также работу с событиями мыши и клавиатуры на примере простой библиотеки SFML.

#### Оценка 3

| Задание | Файл | Функция / Место | Что сделать |
|---------|------|----------------|--------------|
| 3.1 | `src/paint_core.cpp` | `drawPixel` | Проверить границы и установить цвет пикселя |
| 3.2 | `src/paint_core.cpp` | `clearCanvas` | Заполнить холст белым цветом (`std::fill`) |
| 3.3 | `src/paint_core.cpp` | `drawBrush` | Нарисовать закрашенный круг заданного радиуса |
| 3.4 | `src/paint_core.cpp` | `drawRectangle` | Нарисовать прямоугольник по двум углам (`std::min`/`max`) |
